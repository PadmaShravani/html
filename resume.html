Name of the project work  :	FIRE FIGHTING ROBOT FOR FIRE ACCIDENT RESCUE


Description of the project:

The main aim of this project is to detect the fire using temperature sensor and giving an alert with the help of a buzzer.It uses IR sensor to sense the obstacle and send the obstacle distance signal to embedded controller. The embedded controller receives these obstacle detecting signals and then calculates the control signals to drive the motors to change the direction for the robot and the hence the robot is controlled automatically what we can called as a free running robot. The temperature sensor values are given as input to the microcontroller. Microcontroller reads the values from temperature sensor, whenever the temperature exceeds the set point microcontroller gives a buzzer as a fire alert. The main advantage is using this we can provide safety for objects. The applications are in Industrial areas, hotel environments, factory environments and in costly environments.
